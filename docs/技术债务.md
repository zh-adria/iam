# ponytail-debt: 已知简化与升级路径

> 版本：1.1.0 · 更新：2026-07-05

| 项 | 当前 | 升级路径 |
|----|------|----------|
| 单 Maven 模块 | 4 个包代替 Cola 多模块 pom | 团队规模 > 5 人时拆 `iam-adapter/app/domain/infrastructure` 子模块 |
| 实体直落基础设施 | 跳过 Cola 的 Domain → DO 转换 | 高性能场景补 `Assembler` 做对象映射 |
| JWT RS256 | 已落地 RSA keypair | 密钥走 KMS + 定期轮换 |
| OAuth2 端点 | 完整实现（4 grant + PKCE + ID Token） | 引入 `spring-security-oauth2-authorization-server` 进一步标准化 |
| SAML 2.0 | 完整实现（多 IdP + metadata 自动刷新 + 证书） | Signed AuthnRequest / EncryptedAssertion |
| LDAP | 完整实现（多租户 + Group→Role 映射） | 引入 `spring-ldap-ldif-core` + `LdapTemplate.search()` 做组查询 |
| 多租户隔离 | SHARED 模式 | Schema-per-tenant 用 Hibernate `MultiTenantConnectionProvider` |
| ABAC SpEL | 完整实现 | 在 SecurityConfig 用 `MethodSecurityExpressionHandler` 注入自定义 root |
| 社交登录 | 4/5 实现（alipay RSA2 待补） | 接 `OAuth2ClientFilter`，补 alipay RSA2 signing |
| 动态配置 | 完整实现（system_config 表 + Redis pub/sub + 内存缓存） | 无 |
| Session Cookie | 仅 JWT | 需要时引入 Spring Session + Redis |
| SCIM 2.0 | 完整实现（Users/Groups + Bearer Token + filter） | Bulk 批量接口 / SearchRequest POST / ETag |
| SCIM Provisioner Token | 管理态 CRUD + BCrypt 哈希 | 支持 token 过期时间 + 自动轮换 |
| WebAuthn | 仅 stub | 需 Spring Boot 3.x（Spring Security 6.x 原生支持） |
| Kerberos | 仅 stub | 需 `spnego` 库 + KDC 集成 |
