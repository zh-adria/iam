# IAM 平台 — 现状梳理

> 日期：2026-07-07 · 基于当前工作区

---

## 1. 项目边界

### 本期范围（已落地）

| 域 | 内容 |
|---|---|
| 认证协议 | 账密+JWT、MFA TOTP、OAuth2(4 grant+PKCE)、OIDC、SAML 2.0(多IdP+metadata刷新)、LDAP/AD(多租户+Group→Role)、社交(4/5)、CAS 2.0、SMS、Magic Link、SCIM 2.0 |
| 授权 | RBAC + ABAC(SpEL) + API/MENU/BUTTON/DATA 资源类型 |
| 多租户 | SHARED + SCHEMA_PER_TENANT(路由已留口，DDL 自动化未做) |
| 安全 | BCrypt、失败锁定、限流、JWT RS256+JWKS、RefreshToken 轮换、SHA-256 审计链、CORS |
| 管理态 | 用户/角色/权限/客户端/租户/LDAP 映射/审计/配置/SAML IdP/SCIM Token/LDAP Group Mapping API |
| 前端 | Vue3 + Element Plus 管理控制台 9 个 pane |
| 运维 | 加密配置(ENC())、动态配置(DynamicConfig+Redis pub/sub)、Docker Compose |

### 不在本期（YAGNI）

WebAuthn/Kerberos(stub)、Schema DDL 自动化、RS256 热轮换(需 KMS)、SMS/Magic 真实通道、支付宝 RSA2、FIDO2、用户自助改资料、SCIM Bulk/ETag。
